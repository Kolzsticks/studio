# **App Name**: Brava Health

## Core Features:

- Real-Time Temperature Monitoring: Display real-time temperature readings from 8 sensors, showing current highest and lowest temperatures, and visual alerts for abnormal conditions.
- Temperature Trend Analytics: Generate daily, weekly, and monthly temperature trend graphs with historical data and export options.
- AI-Powered Personalized Health Tips: AI tool that uses sensor data to provide personalized health management tips and breast health education content, tailored to the user's specific condition.
- Expert Consultation Module: Facilitate online hospital registration and remote video/chat consultation with doctors.
- Personal Settings & Alerts: Allow users to manage their profile, set temperature alert thresholds, and manage system preferences.
- Third-Party Integration: Enable connection to hospital, community, and family portals, and sync data to cloud storage.

## Style Guidelines:

- Primary color: Soft teal (#70CBC9) for a medical and calming feel.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Light blue (#A7D9ED) to complement the teal and add visual interest.
- Body font: 'PT Sans', a humanist sans-serif for body text and headlines, combining a modern look and a little warmth
- Code font: 'Source Code Pro' for displaying code snippets.
- Use `lucide-react` icons for consistency across the app.
- Implement smooth animations for data updates and transitions to enhance user experience.
